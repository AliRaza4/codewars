###Description:

Create a function christmasTree(height) that returns a christmas tree of the correct height

christmasTree(5) should return:

```
    *    
   ***   
  *****  
 ******* 
*********
```

Height passed is always an integer between 0 and 100.

Use \n for newlines between each line.

Pad with spaces so each line is the same length. The last line having only stars, no spaces.

###My solution:

```javascript
function christmasTree(height) {
  var branch = tree = fill = '';
  var base = 2 * (height - 1) + 1;
  for(var i = 0; i <= height - 1; i++){
    var needles = 2 * i + 1;
    branch = new Array(needles + 1).join('*');
    fill = new Array((base - needles) / 2 + 1).join(' ');
    tree += fill + branch + fill;
    tree += i < height - 1 ? '\n' : '';
  }
  return tree;
}
```
